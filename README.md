# E01-language

## E01 – Minimal Language for Gaming

## Philosophy

E01 is a minimal language.

Word small. Meaning big.

Brain free communication.

The goal is simple communication for games and online chat.

## Example

enemy north  
run  
wait  
team all attack  

enemy dead  
we win  
life good  

---

## Basic Rules

Short sentences  
No complex grammar  

One line = One message

Write short lines.
Do not write long sentences.

Example:

enemy north
3 enemy
danger
team attack
ok

Question: `?`  
Command: `!`  

Plural: add `s`  
Negative: use `no`

Example:

enemy here?  
enemy 3  
no run  
team all attack!

## Respect for English

English0.11 is based on English words and simple grammar.

This project is not meant to replace English.

English is a powerful and beautiful language used all over the world.

E01 is only a small tool for fast and simple communication in games and online chat.

If E01 helps someone become interested in learning English, that would make us very happy.

## Version Rule

E01 versions approach English but never become English.

Example:

0.19

## Numbers

Numbers may always be used.

Example

1 enemy  
3 team  
10 apple  

Numbers are universal and not counted in the 128 word limit.

## Repeat Rule

Words may repeat to increase meaning.

Examples

big big = very big  
danger danger = very dangerous  
run run = run now

## Join Rule

Words may join to make new meaning.

Examples

enemy house = enemy base
team help = support
danger north = north is dangerous
target enemy = attack this enemy

## E01 Grammar Summary

Word order is flexible.

Examples
