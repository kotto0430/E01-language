# E01-language

## E01 – Minimal Language for Gaming

Word small. Meaning big. Brain free.

E01 is a minimal language designed for fast communication in games.  
Players from any country can use it.

---

## Example

enemy north  
run  
wait  
team all attack  

enemy dead  
we win  
life good  

---

## Basic Rules

Short sentences  
No complex grammar  

Question: `?`  
Command: `!`  

Plural: add `s`  
Negative: use `no`

Example:

enemy here?  
enemy 3  
no run  
team all attack!  

---

## Core Idea

Word small.  
Meaning big.  
Brain free.

---

## Version Rule

E01 must never approach English.

Versions extend the decimal part only.

Examples:

E01  
E01.1  
E01.11  
E01.12  
E01.19  
E01.191  

E02 is forbidden.  
E09 is forbidden.

---

## Philosophy

E01 is not meant to replace English.  
It is a small language for fast communication in games.

Word small.  
Meaning big.  
Brain free.
